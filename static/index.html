FAQ

  • Fuwa.se is a simple file hoster, inspired by pomf.se
  • Files may be uploaded via the web interface or by sending a POST request to fuwa.se/api/upload with the file[] key.
  • The max file size for now is 50 MiB, though this may increase later.
  • You may use the short link or the long one with the same effect.
  • Don't upload anything illegal, this includes malware and child porn.
  • Executable files with certain extensions will not be allowed. See this article for reference.
  • Viruses will be scanned for and removed on a regular basis
  • To request the removal of a file, please email abuse@fuwa.se
